**WARDEN Driver Control Scheme**
Below is the current, implemented control scheme for WAR's 2025 robot. Last updated 3/10/26.

### Left Stick:
**Controls:** x/y translation
### Right Stick:
**Controls:** rotation <br>
<br>
### Left Trigger:
**Controls:** deploy/run intake
### Right Trigger:
**Controls:** shoot pipeline <br>
<br>
### Left Bumper:
**Controls:** zero gyro (do not touch ever)
